FAQ & BANTUAN

Bagaimana cara membeli script? ▼
Isi form order di atas atau langsung hubungi kami via WhatsApp/Discord. Setelah konfirmasi produk, lakukan pembayaran, dan script akan dikirim dalam 1-24 jam via DM Discord atau email.
Apakah ada garansi dan update gratis? ▼
Ya! Semua produk mendapat garansi bug fix gratis. Update konten/fitur gratis selama 6 bulan untuk paket Developer, dan selamanya untuk paket Studio.
Metode pembayaran apa saja yang diterima? ▼
Kami menerima QRIS, Dana, OVO, GoPay, Transfer BCA, dan Transfer Mandiri. Semua transaksi aman dan terkonfirmasi secara manual oleh tim kami.
Apakah bisa request fitur custom? ▼
Tentu! Paket Studio sudah termasuk 1x custom request gratis. Untuk paket lainnya, custom request tersedia dengan biaya tambahan tergantung kompleksitas fitur. Hubungi kami untuk konsultasi.
Apakah script aman dari anti-cheat Roblox? ▼
Semua script kami adalah server-sided dan mengikuti ToS Roblox. Script didesain agar kompatibel dengan sistem keamanan Roblox dan tidak melanggar kebijakan platform.
Berapa lama script dikirim setelah pembayaran? ▼
Script biasanya dikirim dalam 1-6 jam setelah pembayaran dikonfirmasi. Maksimal 1x24 jam. Jika ada keterlambatan, kami akan proaktif mengabari kamu via WhatsApp atau Discord.
Apakah ada refund jika script tidak sesuai? ▼
Kami tidak menyediakan refund setelah script dikirim, namun kami memberikan garansi revisi penuh jika ada bug atau script tidak berfungsi sesuai deskripsi. Kepuasan kamu adalah prioritas kami.
